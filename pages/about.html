The Developer Behind the Portfolio

I started my Computer Science journey at the University of Swabi in October 2023 with no freelance clients and a lot of curiosity about what real-world software looked like. By early 2024, I landed my first project-Muhammad Dawood's DevOps portfolio-and since then I've been building full-stack MERN products and designing UI/UX experiences for clients who need practical digital solutions. I care about writing code that's easy to read, creating interfaces that feel clear, and delivering projects on timelines that make sense.

I'm still learning every day, whether it's refining scalable architectures, experimenting with bold design choices, or improving how I collaborate with clients. Each project adds to my experience, and I see every delivery as both a solution and a lesson. My journey is ongoing, and I'm excited to keep growing into a developer who balances technical depth with creative clarity.

How I Think About Work

Code as Craft

Clean, readable, maintainable - always preferred over clever. Good code is code your future self can still understand.

Design is Not Decoration

Every visual decision serves user clarity. If it doesn't make the product easier to use, it doesn't belong in the interface.

Full-Stack Thinking

Every layer matters - from database schema to button hover state. Strong products are built by developers who understand the full picture.
